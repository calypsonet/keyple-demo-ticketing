# Images Directory

Place your image resources here.

## Required Images

- **logo.png** : Logo Calypso/Keyple (displayed on splash screen)
- **icon.png** : Application icon

## Recommendations

- Use PNG format for transparency
- Recommended logo size: 256x256px
- Recommended icon size: 512x512px (will be scaled automatically)
